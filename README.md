# DOCX Text Replacer

Программа для замены текста в файлах DOCX с сохранением форматирования.

## Возможности

- Выбор папки с файлами DOCX
- Ввод текста для поиска и замены
- Поддержка вставки из буфера обмена
- Сохранение форматирования (шрифт, размер, выравнивание)
- Поиск без учета регистра
- Обработка переносов строк и пробелов

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Использование

1. Запустите программу:
   ```
   python src/main.py
   ```

2. Выберите папку с файлами DOCX
3. Введите или вставьте текст для поиска
4. Введите или вставьте текст для замены
5. Нажмите "Заменить текст во всех файлах"

## Компиляция в Windows

Для создания исполняемого файла используйте PyInstaller:

```
pip install pyinstaller
pyinstaller --onefile --windowed src/main.py
```

Исполняемый файл будет создан в папке `dist`.
